# Trade Settlement Prediction - ML Project

## Quick Start

### Run ML Pipeline
```bash
python3 run_pipeline.py
```

### Launch Dashboard
```bash
streamlit run app.py
```

## Project Structure
- `Data/` - Input CSV files
- `src/` - Source code modules
- `models/` - Trained models
- `results/` - Output reports and metrics
- `dashboard/` - HTML dashboards (use app.py for interactive)
- `app.py` - Streamlit dashboard (recommended)
